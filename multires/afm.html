AFM Image of Semiconductor Quantum Dots

  • Input image. Image from Ian Farrer, if10004@cam.ac.uk, (further information).
  • List of objects detected, with properties. Postscript file (derived from LaTeX table).
  • Map of object positions. Postscript.
  • Map of sub-object positions. Postscript.
  • Command used:
    mr_detect -i 2 -n 3 -K -g 1 image3.fits image3out
    where from the input image, image3.fits, files image3out_obj.ps, image3out_subobj.ps and image3.tex, among others are produced, together with a noise filtered image, image3out.fits. Parameters are: -i 2, only use 2 iterations to refine the "photometric" properties measured (not relevant here); -n 3, use a small number of resolution scales, 3; -K, take account of "objects" close to the image boundary; -g 1, use just one standard deviation to define the signal signficance level.
    See MR/1 User Manual, online HTML (limited) version.
