Various Prototypes

Text associated with images, or class characterizations of coordinates related to images (based only on spectra here!), are used for - in some measure - content-based image retrieval.
  • Query HST Image Database using text and associated images. Note how texts and images have been set up for this. The data used here is not updated automatically at present. The image links are built into the text which is indexed. An example at the end shows retrieval using 'externally' associated text and image data. Link to ESO. Requires WAIS. Not working?
  • HST Proposal Abstracts Search Form. Free text searching using freeWAIS-sf and SFgate. Fields (Proposal ID, PI) are supported. Pointers to freeWAIS-sf documentation. Link to ESO. Requires freeWAIS-sf. Try e.g. as PI name: Walsh. Try e.g. as abstract text: herbig haro.
  • Query-by-class, HST Image Database. Spectra (FOS, GHRS) have been classified by (i) matching by coordinate with IUE objects, and (ii) text analysis of object name, or title/abstract of proposal. Link to ESO.
